Course Description

  • Covers scrum as an industry standard methodology.
  • Apply test driven development, version control, continuous integration and continuous delivery in blockchain
  • Best practices in DevOps using industry standard tools

Evaluation System

Assessment Description Weightage
Participation In class participation 10%
Lab Exercises 5 @6% Mock sprints + Role Evaluation 30%
Lab Tests 5 @6% DevOps activities 30%
Final Exam Short answer 30%

Learning outcome

  • Apply Scrum methodology to effectively participate in the management of a development project
  • Utilize advanced features of a version control tool such as Git
  • Effectively participate in the project development process using a test-driven approach
  • Deploy smart contract to a test network
  • Understand and implement a Continuous Integration/Continuous Delivery pipeline
